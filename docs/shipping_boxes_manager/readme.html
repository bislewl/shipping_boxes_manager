Title:Shipping Boxes Manager
Author:Numinix
Description:This module creates a management system where Zen Cart store admins can define shipping boxes. The dimensions and weights assigned to each box are then used to determine the most accurate shipping rates in supported modules.

This module is currently compatible with:

  • Endicia XML Exporter / Importer
  • FedEx Web Services Shipping
  • UPS United Parcel Service
  • USPS United States Postal Service
Installation Level:Moderate
Installation Time:~ 10 minutes

INSTALLATION
  1. Backup You database & files
  2. Install Numinix Product Fields with the optional Dimensions fields and assign dimensions to each of your products (to add dimensions to attributes, also install Attribute Dimensions).
  3. Patch your database by copying and pasting the contents of install.sql into ADMIN->TOOLS->INSTALL SQL PATCHES.
  4. Upload all file contained in /catalog/ to your store's root while maintaining your directory structure.
  5. Go to ADMIN->TOOLS->SHIPPING BOXES MANAGER and add all of your shipping boxes. You may want to add extra space to the height of the box to account for flex (if box is overfilled).
  6. Go to ADMIN->CONFIGURATION->SHIPPING BOXES MANAGER CONFIGURATION and enable the module.
  7. Optional: If you would like to use package dimensions in your rate quotes, upload the ups.php and usps.php modules. Refer to these files for edits that would need to be made to other third party mods in order to support dimensions.
TIPS
UNINSTALL
  1. Remove edits performed in the installation procedure above.
  2. Delete all uploaded files from your store.
  3. Patch your database by copying and pasting the contents of uninstall.sql into ADMIN->TOOLS->INSTALL SQL PATCHES
